这是打磨了几个月的网页。最近，有人看了觉得"不够 Fluent"，"太紧了"，觉得 Fluent Design 就是 8px 圆角与窄间距，我认为这是不恰当的。

我想谈谈我对 Design 的理念。

首先，Fluent Design 从 1.0 时代就已划分了 Web、Android、iOS 与 Windows 等多个平台场景，每个平台的设计多多少少都有变化，包括 Web 端。

很多设计系统在 Guide 里也是按平台拆开写的，同一套控件，很多用法也做不到完全统一。举个例子：Apple Music 在 Windows 平台用 WinUI 3 做界面，和别的平台上也不会长得一模一样，看起来也很原生，很舒服。

再谈 Web，我对于这方面的设计理念是：布局比 Desktop 端的间距大一些，留白多一些，可读性强。没素材用 Card，圆角大一些。很多知名页面的设计都遵循了这一套理念。

最后说到 Fluent，至少官方自己的 Fluent 2 页面也并不是窄间距、小圆角，甚至 Button 是药丸式的。下面是一张 Hero 区截图，可以看看。
